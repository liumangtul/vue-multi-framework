# demo1

## Project setup
```
npm install
```
### 1、创建项目目录
```
    node create.js [项目名称]/[版本号]
    node create.js demo/v1.0.0
```

### 2、创建页面模板
```
    cd src/demo
    node createpage.js page
    or
    node createpage.js page1,page2,...
```

### Compiles
```
path:vue-project
  npm run build demo 打包到trunks下
  npm run watch demo 打包到branches下 
 
```
